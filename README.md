# coboView
產學專案

## 簡介
1.使用html 已CDN的方式已入vue vuetify 製作網站  
2.javascript 製作前端的新增、刪除、查詢、修改、拖曳、父傳子、子傳父狀態等功能  
3.RWD符合電腦螢幕、平板大小  
4.使用 vue-i18n 進行多國語系轉換  

## 使用技術
vue.js  
vuetify  
html  
css  
javacrtipt
